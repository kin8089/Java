비지도 학습

1. 데이터의 비지도 변환 : 데이트를 새롭게 표현하여 사람이나 알고리즘이 기존 데이터보다 쉽게 해석

-> 차원 축소분야에서 널리 쓰임 (시각화를 위해 데이터셋을 2차원으로 변경)
    데이터를 구성하는 단위나 성분을 찾기도 함 (문서에서 주제 찾고 학습)
    소셜미디어에서 사진 분류
    
-> 알고리즘이 유용한 내용을 학습했는지 평가가 어려움
    레이블이 없는 데이터에 적용하기 때문에 무엇이 올바른지 모름
    학습의 결과를 확인할 때는 직접 확인해야 할 때가 많음



2. 군집 : 데이터를 클러스터라는 그룹으로 나누는 작업

-> k-평균 군집 : 가장 간단하고 널리 사용하는 알고리즘
		데이터의 어떤 영역을 대표하는 클러스터 중심을 찾음
		데이터 포인트를 클러스터 중심에 할당 -> 데이터 포인트의 평균으로 다시 중심 지정
		데이터 포인트의 변화가 없을때까지 반복한다

	군집은 각 데이터 포인트가 레이블을 가진다는 면에서 분류와 비슷함
	정답을 모르고 있어서 레이블 자체에 의미는 없음
	초기화를 무작위로 하기 때문에 클러스터 번호는 고정되지 않음
