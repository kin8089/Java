-- vi
지정한 파일을 읽어오고 파일을 지정하지 않으면 빈 파일이 열림
* a / i - 입력모드(문서작성)
* (:) 콜론 - 라인모드(저장,종료,취소), w : 저장 / wq : 저장하고 종료 / w[파일명] : 새 파일로 저장 / q : 종료 / q! : 저장하지않고 종료
* 명령모드에서 - /문자열 : 찾을 문자열을 커서의 위치에서 아래쪽으로 검색
                   - ?문자열 : 찾을 문자열을 커서의 위치에서 위쪽으로 검색

* :s/문자열1/문자열2/ - 커서가 위치한 행에서 첫번째로 나오는 문자열1을 문자열2로 치환
  :%s/문자열1/문자열2/g - 파일 전체에서 모든 문자열1을 문자열2로 치환
  :<범위>s/문자열1/문자열2/ - 범위 내 모든 행의 각행에서 첫번째로 검색되는 문자열1을 문자열2로 치환

* u - 명령을 취소
  U - 해당 행에서 수행한 모든 명령을 취소
  :e! - 마지막으로 저장한 내용 이후의 것을 버리고 새로 작업

* x - 현재 커서가 위치한 문자를 삭제
  dw - 단어삭제
  dd - 현재 커서의 행 삭제
  숫자 dd - 현재 커서부터 숫자만큼의 행 삭제
  yy - 현재 커서가 있는 행을 복사
  숫자 yy - 현재 커서부터 숫자만큼의 행을 복사
  p - 복사한 내용을 현재 행 이후에 붙여넣기
  P - 복사한 내용을 현재 행 이전에 붙여넣기

* w - 다음 단어의 처음으로 이동
  ^ - 줄의 첫 문자로 이동
  $ - 줄의 맨 끝으로 이동
  G - 제일 끝 행으로 이동
  gg - 제일 첫 행으로 이동
  숫자 G - 숫자 행으로 이동
